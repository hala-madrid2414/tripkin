# 旅行地图功能 — 前端开发文档

> 文档版本：v2.0
> 适用项目：刷到即同行（抖音 AI 创变者计划黑客松联赛·南京栖霞站）
> 负责模块：旅行地图（唯一页面）
> 更新日期：2026-06-18

---

## 一、需求总览

本页面为 H5 旅行灵感地图，是整个产品的唯一核心页面。

**包含功能：**

1. 展示完整中国地图，支持拖动、放大、缩小
2. 三个数据图层：漂流瓶个数 / 搭子个数 / 热门景点个数，用颜色深浅表示数量
3. 点击省份区域聚焦放大，展示该省下属景点
4. 点击景点弹出底部卡片（简介+标签+三个操作按钮），卡片可上拉展开
5. 搜索地点并跳转定位

**不做功能：**

- 卡片上拉后的详情内容（UI 框架做出来，内容留空）
- 真实后端接口（全部使用 Mock 数据）
- 定位、路线规划、导航

---

## 二、技术方案

### 2.1 推荐方案：SVG 中国地图 + CSS Transform 手势

```
地图底图：简化版 SVG 中国省级行政区划（34个省级 path）
缩放拖拽：CSS transform: scale() + translate() + touch 事件
热力展示：SVG circle 元素，通过 radius 和 fill-opacity 表示数量
标记点位：SVG foreignObject 或 g 元素内嵌 HTML
```

**为什么选这个方案：**

- 包体极小（SVG 仅 ~30KB，无需下载 tile 图）
- 风格完全可控（匹配设计稿的浅色清新风）
- 渲染性能好（H5 端几十个省+几十个标记无压力）
- 不需要申请地图密钥

### 2.2 核心依赖

| 库               | 用途                                | 安装 |
| ---------------- | ----------------------------------- | ---- |
| 无需第三方地图库 | —                                   | —    |
| 手势自行实现     | touchstart/move/end + CSS transform | 内置 |

可选辅助库（如果团队熟悉可以用）：

- `panzoom` — SVG/元素平移缩放库（`npm install panzoom`）

## 三、Mock 数据结构

### 3.1 类型定义（map.types.ts）

```typescript
/** 省份数据 */
export interface Province {
  id: string // 唯一标识，如 "sichuan"
  name: string // 显示名称，如 "四川"
  path: string // SVG path 数据
  center: { x: number; y: number } // 中心点坐标（SVG 坐标系内）
  // 三个图层的数据
  bottleCount: number // 漂流瓶数量
  companionCount: number // 搭子数量
  hotspotCount: number // 热门景点数量
  // 下属景点
  spots: Spot[]
}

/** 景点数据 */
export interface Spot {
  id: string // 唯一标识，如 "chuanxi"
  name: string // 名称，如 "川西"
  subtitle?: string // 副标题，如 "318沿线"
  parentId: string // 所属省份 id
  position: { x: number; y: number } // 在地图上的坐标（SVG 坐标系）
  bottleCount: number
  companionCount: number
  hotspotCount: number
  tags: string[] // 标签，如 ["雪山", "自驾", "美食"]
  description: string // 景点简介
}

/** 图层类型 */
export type LayerType = 'bottle' | 'companion' | 'hotspot'

/** 地图视口状态 */
export interface Viewport {
  scale: number // 缩放比例，默认 1
  x: number // 平移 X，默认 0
  y: number // 平移 Y，默认 0
}

/** 地图状态 */
export interface MapState {
  viewport: Viewport
  activeLayer: LayerType
  selectedProvince: string | null // 选中的省份 id
  selectedSpot: string | null // 选中的景点 id
  cardExpanded: boolean // 卡片是否展开
  isLayerPanelOpen: boolean // 图层面板是否打开
  searchQuery: string
  searchResults: Spot[]
}
```

// 数量 → 半径映射（归一化）
const getRadius = (count: number, max: number): number => {
  const minR = 20,
    maxR = 55
  return minR + (count / max) * (maxR - minR)
}

// 数量 → 透明度映射
const getOpacity = (count: number, max: number): number => {
  const minO = 0.2,
    maxO = 0.6
  return minO + (count / max) * (maxO - minO)
}

// useMapGesture.ts 中的卡片手势逻辑
const useCardGesture = ({
  onExpand,
  onCollapse,
  onClose,
}: {
  onExpand: () => void
  onCollapse: () => void
  onClose: () => void
}) => {
  const touchStartY = useRef(0)
  const currentTranslateY = useRef(0)

  const handleTouchStart = (e: TouchEvent) => {
    touchStartY.current = e.touches[0].clientY
  }

  const handleTouchMove = (e: TouchEvent) => {
    const deltaY = e.touches[0].clientY - touchStartY.current
    // 向上滑（负值）→ 展开
    // 向下滑（正值）→ 收起/关闭
    currentTranslateY.current = deltaY
  }

  const handleTouchEnd = () => {
    const deltaY = currentTranslateY.current

    if (deltaY < -80) {
      // 上滑超过阈值 → 展开
      onExpand()
    } else if (deltaY > 100) {
      // 下滑超过阈值 → 关闭
      onClose()
    } else if (deltaY > 60) {
      // 下滑较小 → 收起
      onCollapse()
    }
    currentTranslateY.current = 0
  }

  return { handleTouchStart, handleTouchMove, handleTouchEnd }
}

### 6.1 地图手势

| 手势         | 效果      | 实现方式                                      |
| ------------ | --------- | --------------------------------------------- |
| 单指拖动     | 平移地图  | touchstart/move 记录 delta，更新 translateX/Y |

| 点击省份     | 聚焦该省  | 动画过渡到省份中心，显示景点标记              |
| 点击景点     | 弹出卡片  | 底部卡片从 hidden → collapsed                 |
| 点击地图空白 | 关闭卡片  | 卡片关闭，可选手动关闭省份聚焦                |

**缩放范围：**

- 最小缩放：`scale: 0.8`（全国视图略缩小）
- 最大缩放：`scale: 4`（聚焦到城市级别）
- 默认缩放：`scale: 1`

### 6.2 卡片手势

| 手势       | 当前状态  | 结果                 |
| ---------- | --------- | -------------------- |
| 向上滑动   | collapsed | 展开卡片（expanded） |
| 向下滑动   | expanded  | 回到 collapsed       |
| 向下滑动   | collapsed | 关闭卡片（hidden）   |
| 点击拖拽条 | collapsed | 展开卡片             |
| 点击拖拽条 | expanded  | 收起卡片             |

**动画参数：**

- 展开过渡：height 240px → 85vh，350ms，cubic-bezier(0.25, 0.46, 0.45, 0.94)
- 关闭：`transform: translateY(0) → translateY(100%)`，250ms，ease-in

type MapAction =
  | { type: 'SET_VIEWPORT'; payload: Partial<Viewport> }
  | { type: 'SELECT_PROVINCE'; payload: string }
  | { type: 'SELECT_SPOT'; payload: string }
  | { type: 'CLEAR_SELECTION' }
  | { type: 'EXPAND_CARD' }
  | { type: 'COLLAPSE_CARD' }
  | { type: 'CLOSE_CARD' }
  | { type: 'SET_LAYER'; payload: LayerType }
  | { type: 'TOGGLE_LAYER_PANEL' }
  | { type: 'SET_SEARCH'; payload: string }
  | { type: 'RESET_TO_NATIONAL' }

const initialState: MapState = {
  viewport: { scale: 1, x: 0, y: 0 },
  activeLayer: 'bottle',
  selectedProvince: null,
  selectedSpot: null,
  cardExpanded: false,
  isLayerPanelOpen: false,
  searchQuery: '',
  searchResults: [],
}

const mapReducer = (state: MapState, action: MapAction): MapState => {
  switch (action.type) {
    case 'SELECT_PROVINCE': {
      const prov = findProvinceById(action.payload)
      if (!prov) return state
      // 计算聚焦所需的 scale 和 translate
      const targetScale = 2.5
      const containerW = window.innerWidth
      const containerH = window.innerHeight
      const targetX = containerW / 2 - prov.center.x * targetScale
      const targetY = containerH / 2 - prov.center.y * targetScale
      return {
        ...state,
        selectedProvince: action.payload,
        selectedSpot: null,
        viewport: { scale: targetScale, x: targetX, y: targetY },
        cardExpanded: false,
      }
    }
    case 'SELECT_SPOT':
      return { ...state, selectedSpot: action.payload, cardExpanded: false }
    case 'EXPAND_CARD':
      return { ...state, cardExpanded: true }
    case 'COLLAPSE_CARD':
      return { ...state, cardExpanded: false }
    case 'CLOSE_CARD':
      return { ...state, selectedSpot: null, cardExpanded: false }
    case 'RESET_TO_NATIONAL':
      return {
        ...state,
        selectedProvince: null,
        selectedSpot: null,
        viewport: { scale: 1, x: 0, y: 0 },
        cardExpanded: false,
      }
    // ... 其他 case
  }
}

### 9.1 地图手势（useMapGesture.ts）

```typescript
export const useMapGesture = (
  viewport: Viewport,
  onViewportChange: (v: Viewport) => void,
) => {
  const containerRef = useRef<HTMLDivElement>(null)
  const gestureState = useRef({
    isDragging: false,
    startX: 0,
    startY: 0,
    startViewportX: 0,
    startViewportY: 0,
    lastScale: 1,
    initialPinchDistance: 0,
  })

  // 单指拖拽
  const handleTouchStart = (e: TouchEvent) => {
    if (e.touches.length === 1) {
      gestureState.current.isDragging = true
      gestureState.current.startX = e.touches[0].clientX
      gestureState.current.startY = e.touches[0].clientY
      gestureState.current.startViewportX = viewport.x
      gestureState.current.startViewportY = viewport.y
    }
    // 双指缩放
    if (e.touches.length === 2) {
      const dx = e.touches[0].clientX - e.touches[1].clientX
      const dy = e.touches[0].clientY - e.touches[1].clientY
      gestureState.current.initialPinchDistance = Math.sqrt(dx * dx + dy * dy)
      gestureState.current.lastScale = viewport.scale
    }
  }

  const handleTouchMove = (e: TouchEvent) => {
    e.preventDefault() // 防止页面滚动

    if (e.touches.length === 1 && gestureState.current.isDragging) {
      const dx = e.touches[0].clientX - gestureState.current.startX
      const dy = e.touches[0].clientY - gestureState.current.startY
      onViewportChange({
        ...viewport,
        x: gestureState.current.startViewportX + dx,
        y: gestureState.current.startViewportY + dy,
      })
    }

    if (e.touches.length === 2) {
      const dx = e.touches[0].clientX - e.touches[1].clientX
      const dy = e.touches[0].clientY - e.touches[1].clientY
      const distance = Math.sqrt(dx * dx + dy * dy)
      const scaleFactor = distance / gestureState.current.initialPinchDistance
      const newScale = Math.max(
        0.8,
        Math.min(4, gestureState.current.lastScale * scaleFactor),
      )
      onViewportChange({ ...viewport, scale: newScale })
    }
  }

  const handleTouchEnd = () => {
    gestureState.current.isDragging = false
  }

  // 绑定事件
  useEffect(() => {
    const el = containerRef.current
    if (!el) return
    el.addEventListener('touchstart', handleTouchStart, { passive: false })
    el.addEventListener('touchmove', handleTouchMove, { passive: false })
    el.addEventListener('touchend', handleTouchEnd)
    return () => {
      el.removeEventListener('touchstart', handleTouchStart)
      el.removeEventListener('touchmove', handleTouchMove)
      el.removeEventListener('touchend', handleTouchEnd)
    }
  }, [viewport])

  return { containerRef }
}
```

## 十一、性能优化

| 优化点   | 方案                                            |
| -------- | ----------------------------------------------- |
| 渲染优化 | SVG 使用 `will-change: transform` 开启 GPU 加速 |
| 动画流畅 | 所有位移动画使用 CSS transform，不触发布局重排  |

| 懒加载   | 卡片展开后才渲染上拉区域的 DOM                  |
